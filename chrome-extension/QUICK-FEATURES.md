## How to Use New Features

### Generate Icons
1. Open `generate-icons.html` in Chrome
2. Customize text/colors
3. Click "Generate Icons"
4. Click "Download All Icons"
5. Place icons in `chrome-extension/icons/` folder

### Quick Convert
1. Right-click any image on a webpage
2. Click extension icon
3. Click "Quick Convert"
4. Converter opens with that image

### Settings Management
1. Right-click extension icon â†’ Options
2. Configure all settings
3. Click "Save Settings"
4. Use Export to backup
5. Use Import to restore

### Image Counter
- Automatically shows in popup
- Updates when you visit new pages
- Helps plan batch conversions
