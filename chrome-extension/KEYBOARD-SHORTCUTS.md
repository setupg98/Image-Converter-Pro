# Keyboard Shortcuts

## Extension-Wide Shortcuts

### In Converter Interface
- `Ctrl/Cmd + U` - Upload files
- `Ctrl/Cmd + Enter` - Convert all images
- `Ctrl/Cmd + Z` - Clear all
- `Ctrl/Cmd + D` - Download all
- `Ctrl/Cmd + H` - View history
- `Esc` - Close modals

### Chrome Extension Specific
- `Alt + I` - Open extension popup (Chrome default)
- `Alt + Shift + I` - Open extension options (Chrome default)

## Customizable Shortcuts

You can customize shortcuts in Chrome:
1. Go to `chrome://extensions/shortcuts`
2. Find "Image Converter Pro"
3. Set custom shortcuts for:
   - Open converter
   - Quick convert
   - Convert page images

## Tips

- Use `Ctrl/Cmd + Enter` for quick conversions
- `Esc` closes any open modals
- `Ctrl/Cmd + D` downloads all converted files
- `Ctrl/Cmd + H` shows conversion history
