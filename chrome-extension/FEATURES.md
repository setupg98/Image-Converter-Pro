## Usage Scenarios

### Scenario 1: Convert Image from Webpage
1. Browse any website
2. Right-click an image
3. Select "Convert Image with Image Converter Pro"
4. Converter opens with image loaded
5. Configure and convert

### Scenario 2: Convert All Images on Page
1. Visit a webpage with multiple images
2. Click extension icon
3. Click "Convert All Images on Page"
4. All images are extracted and loaded
5. Batch convert them all

### Scenario 3: Quick Access
1. Click extension icon in toolbar
2. Click "Open Full Converter"
3. Upload images manually
4. Use all features

### Scenario 4: From Anywhere
1. Right-click anywhere on a page
2. Select "Open Image Converter Pro"
3. Full converter interface opens
4. Drag & drop or upload images
