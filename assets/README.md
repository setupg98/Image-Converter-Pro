# Assets Folder

Place application icons here:

- `icon.png` - Linux icon (512x512 PNG)
- `icon.ico` - Windows icon (multi-size ICO)
- `icon.icns` - macOS icon (ICNS format)

## Creating Icons

You can use online tools or image editors to create icons:

1. Create a 512x512 PNG image
2. For Windows: Convert to ICO format (use online converter or ImageMagick)
3. For macOS: Convert to ICNS format (use online converter or iconutil)

## Default Behavior

If icons are not provided, Electron will use default icons. The app will still work, but custom icons make it look more professional.
